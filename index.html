Belajar Grid System

Apa sih Grid itu? Penting tidak Grid diterapkan pada website?

Menggunakan Grid System memudahkan kita dalam mengatur tata letak elemen saat membuar website. Zaman dahulu, kebanyakan website menggunakan tag table untuk menngatur tata letak (layout-nya). Sekarang sudah tag table tidak digunakan lagi, karena kurang efektif dan tidak SEO Friendly. Oleh karena itu, website zaman sekarang menggunakan CSS untuk mengatur tata letaknya. Sistem Grid di Bootstrap mengadopsi konsep table. Karena itu, kita hanya perlu menggunakan tiga kelas untuk membuatnya. Kelas tersebut kita terapkan dalam tag 'div'.
Berikut ini tiga kelas yang dimaksud, diantaranya :
1. Kelas .continer : Kelas continer berfungsi membungkus konten website.
Kelas ini sama fungsinya seperti tag 'table'.
2. Kelas .row : Berfungsi untuk membuat baris (penempatannya didalam kelas .container),
3. Kelas .col : Berfungsi untuk membuat kolam (penempatannya didalam kelas .row).
Grid System yang ada di Bootstrap mempunyai fungsi sendiri-sendiri, diantaranya :

  • col-lg-* : digunakan untuk mengatur grid pada monitor komputer yang berukuran besar.
  • col-md-* : digunakan untuk mengatur grid pada monitor komputer yang berukuran sedang.
  • col-sm-* : digunakan untuk mengatur grid pada monitor yang berukuran tablet.
  • col-xs-* : digunakan untuk mengatur grid untuk ukuran handphone.
Untuk keterangan lebih lanjut silahkan jenengan bisa baca-baca di dokumentasi yang sudah dibuat Bootstrap disini. Jangan lupa gunakan search docs... agar lebih cepat (gunakan bahasa inggris).
